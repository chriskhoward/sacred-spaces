ABOUT THE WEBSITE

Flow in Faith Teachers Collective is a community-powered platform created exclusively for Christian Yoga Teachers of Color. The website serves as a dedicated space where teachers and practitioners can connect, grow, and support one another while honoring their faith, culture, and calling.

USER AUTHENTICATION & MEMBERSHIP

The website uses Clerk for secure user authentication. There are two types of members:

1. TEACHERS
   • Access to teaching resources
   • Directory profile management
   • Community calls participation
   • Teaching opportunities

2. PRACTITIONERS
   • Video library access
   • Live class schedules
   • Directory search to find teachers
   • Community engagement

When users first sign up, they go through an onboarding process to select their membership type (Teacher or Practitioner). This determines what features and content they can access.
